📋 UI 设计说明

点击数字标记查看详情

#header-section
1. 顶部导航栏
设计说明：
- 采用深色渐变背景，营造科技感
- 左侧Logo：包含图标和品牌文字，使用Orbitron字体
- 中间导航：资讯看板/运行监控页面切换
- 右侧信息：最后更新时间 + 手动刷新按钮

交互说明：
- 导航Tab点击切换页面，带淡入动画
- 刷新按钮点击触发数据更新
#nav-tabs
2. 页面切换导航
设计说明：
- 采用胶囊式设计，置于半透明容器中
- 激活状态：霓虹蓝填充 + 发光效果
- 未激活状态：透明背景 + 灰色文字

交互说明：
- 悬停：背景变亮，文字变白
- 点击：切换页面，状态平滑过渡
#platform-tabs
3. 平台筛选Tab
设计说明：
- 下划线式Tab设计，简洁清晰
- 每个Tab显示平台名称 + 资讯数量徽章
- 激活状态：霓虹蓝文字 + 霓虹蓝底部边框

交互说明：
- 悬停：背景变亮
- 点击：筛选对应平台资讯
- 全部：显示所有平台内容
.news-card
4. 资讯卡片
设计说明：
- 左侧封面：16:9比例，占位或真实图片
- 平台徽章：渐变色设计，区分不同平台
- 标题：2行截断，避免过长
- 元数据：发布时间 + 平台专属指标（阅读/播放/点赞等）
- 标签：#号前缀，圆角设计

交互说明：
- 悬停：右侧发光，向左滑出4px，左侧渐变条出现
- 点击：新标签页跳转到原始链接
.metric-card
5. 指标卡片
设计说明：
- 渐变背景 + 半透明装饰效果
- 数值：等宽字体，大尺寸显示
- 趋势指示器：上下箭头 + 背景色

颜色规则：
- 成功/上升：绿色 (#22C55E)
- 警告/中性：黄色 (#F59E0B)
- 错误/下降：红色 (#EF4444)
.platform-status-card
6. 平台状态卡片
设计说明：
- 状态圆点：在线绿色/离线红色，带霓虹发光效果
- 在线状态：绿色脉冲动画，呼吸效果
- 边框颜色与状态同步

动画说明：
- 在线圆点：2秒循环呼吸动画
.failure-item
7. 失败原因统计
设计说明：
- 横向条形图，直观展示占比
- 不同失败原因对应不同颜色：
  • 网络超时 → 红色
  • Cookie过期 → 黄色
  • 页面结构变化 → 紫色
  • 认证失败 → 蓝色
  • 其他 → 灰色

交互说明：
- 条形宽度带0.5秒加载动画
.record-item
8. 执行记录项
设计说明：
- 左侧图标：✓成功(绿)/✕失败(红)，圆形背景
- 左侧边框：与状态同色
- 时间戳：等宽字体显示
- 错误信息：红色高亮

交互说明：
- 悬停：背景变亮，展示详情

色彩系统：
• 主色：#60A5FA（霓虹蓝）
• 背景：#09090F（深黑）
• 表面：#12121A（深灰）

字体系统：
• 标题：Orbitron
• 正文：Exo 2
• 代码：JetBrains Mono

• 4px基数递增

• 小元素：4-8px
• 卡片：12px
• 大容器：16px
