🤖 AI 热点资讯监控系统

实时追踪各平台 AI 大模型、应用、Agent 工具最新动态

✨ 核心功能

🔥 热榜优先采集

优先抓取各平台热搜/热榜内容，确保获取最热门资讯。数据源优先级：微博热搜 → B站热门 → 小红书热榜 → X趋势 → Facebook热门

🎯 智能关键词过滤

AI大模型、LLM、GPT、Claude、Agent、AI应用、生成式AI等关键词精准匹配，只看你关心的内容

📊 多平台整合

微博、B站、小红书、X(Twitter)、Facebook 五大平台，一站式查看，无需切换

🖼️ 封面图片展示

自动获取各平台资讯封面，视觉化呈现，提升阅读体验

📈 运行监控看板

实时监控爬虫成功率、响应延迟、采集数量等指标，7天历史数据回溯。包含失败原因统计（网络超时、Cookie过期、页面结构变化等）和最近采集记录列表

⚡ GitHub 自动部署

使用 GitHub Actions 每2小时自动执行，GitHub Pages 托管网页，零服务器成本

🏗️ 技术架构

后端

Python 3.x + BeautifulSoup4 + Requests + Selenium

数据存储

SQLite + JSON + GitHub Pages

前端

HTML5 + CSS3 + JavaScript + ECharts

部署调度

GitHub Actions (cron: 每2小时)

🚀 GitHub 部署方案

✅ GitHub 自动执行

  • ⏰ 每2小时自动触发 GitHub Actions
  • 🔍 执行各平台热榜爬虫
  • 🧹 关键词过滤、去重、热度排序
  • 📸 封面图片获取与处理
  • 📊 生成运行监控数据
  • 💾 导出 data.json
  • 📝 Git commit & push
  • 🌐 GitHub Pages 自动部署

⚙️ 需要人工配合

  • 1. 创建 GitHub 仓库
  • 2. 配置 Secrets（各平台 Cookie）
  • 3. 启用 GitHub Pages
  • 4. 本地开发环境搭建
  • 5. Cookie 更新（按需）
  • 6. 关键词调整（按需）
  • 7. 爬虫修复（页面结构变化时）
  • 8. 监控看板查看（日常运维）
